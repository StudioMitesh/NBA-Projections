NBAProjections.xlsx: File containing the counting stats calculated for 60 of the top young players in the NBA today

Players: Folder containing data on player comparisons, similarity scores to be processed

projections.py: Python file that accumulates data on players, runs a ranking regression algorithm to predict future seasons counting statistics.

sample_pl.csv: csv file that takes in the numpy arrays, which are the output statistics from projections.py.
